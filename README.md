# WAV 音效檔播放器 (WAV Audio Player)

## 📖 專案簡介
這是一個簡易WAV音效檔播放器。

### ✨ 主要功能
* **瀏覽檔案**：透過 `OpenFileDialog` 讓使用者選擇本機的 `.wav` 音效檔。
* **播放控制**：
  * **播放一次**：載入音效並完整播放一次。
  * **重複播放**：循環播放選定的音效檔。
  * **停止播放**：中斷目前正在播放的音效。
* **安全防護**：點擊「結束程式」或關閉視窗時，會跳出對話方塊確認是否真的要結束。

---

## 🚀 執行說明

### 操作步驟
1. 點擊 **「瀏覽」** 按鈕，在彈出的視窗中尋找並選擇一個 `.wav` 音效檔。
2. 選擇檔案後，檔案的完整路徑會顯示在「音效位置」的文字方塊中。
3. 點擊 **「播放一次」** 測試單次播放，或點擊 **「重複播放」** 測試循環功能。
4. 若要中斷聲音，請點擊 **「停止播放」**。
5. 點擊 **「結束程式」** 或視窗右上角的 `X`，系統會跳出「確定要關閉應用程式嗎？」的確認視窗，點選「是(Y)」即可完整退出程式。
